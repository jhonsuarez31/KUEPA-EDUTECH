# KUEPA API

1. Clonar proyecto
2. ```npm install```
3. Clonar el archivo ```.env.development.local```
4. Cambiar las variables de entorno
5. Levantar la base de datos
```
docker-compose up -d
```

6. Levantar: ```npm start:dev```

7. Cargar el archivo kuepa_db.sql
